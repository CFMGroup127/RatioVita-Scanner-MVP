# Agent Memory Document Organization Guide

## Overview
This guide describes the standardized organization system for all agent memory documents using Google Docs tabs, headings, and templates.

## Universal Tab Structure (All Agents)

### 1. Introduction
- **Template:** Profile
- **Purpose:** Agent profile, role description, and basic information
- **Content:**
  - Agent name and role
  - Email address
  - Status (Active/Inactive)
  - Role description and responsibilities

### 2. Tasks
- **Template:** Task Tracker
- **Purpose:** Daily task tracking with date-based subtabs
- **Structure:**
  - Main heading: `## TASKS`
  - Daily subtabs: `### YYYY-MM-DD` or `### Month DD, YYYY`
  - Each day contains:
    - Today's Tasks (checklist)
    - Completed Tasks
    - Notes

### 3. Protocols
- **Template:** Compliance Log
- **Purpose:** Protocol compliance tracking (P0-P12)
- **Content:**
  - Protocol checklist
  - Compliance log entries
  - Timestamps and status

### 4. Meetings
- **Template:** Meeting Notes
- **Purpose:** Meeting acceptances, notes, and participation
- **Content:**
  - Meeting acceptance logs (P8)
  - Meeting notes (P5)
  - Participation records

### 5. Reports
- **Template:** Report Archive
- **Purpose:** Formal reports and submissions
- **Content:**
  - Submitted reports
  - Report status
  - Acknowledgments

## Daily Workflow

### Task Tab Structure
Each day, agents should:
1. Create a new date subsection under Tasks: `### November 20, 2025`
2. List today's tasks in checklist format
3. Move completed tasks to "Completed Tasks"
4. Add notes and observations

### Example Daily Entry
```
## TASKS

### November 20, 2025
**Date:** November 20, 2025

#### Today's Tasks
- [ ] Review meeting agenda
- [ ] Prepare report
- [ ] Send confirmation email

#### Completed Tasks
- [x] Meeting acceptance logged
- [x] P8 protocol completed

#### Notes
- Meeting scheduled for 2:30 PM EST
- Report due tomorrow
```

## Implementation

### Initial Setup
Run `organize_agent_memory_documents.py` to:
1. Create standardized structure for all agents
2. Add role-specific tabs
3. Initialize daily task sections

### Daily Usage
Agents should:
1. Write to appropriate tab/section
2. Use date-based subtabs for tasks
3. Follow template formats
4. Maintain chronological order

### Memory Tool Enhancement
The Google Docs Memory Tool should be enhanced to:
1. Accept section/tab parameters
2. Write to specific sections
3. Create date-based subtabs automatically
4. Format content according to templates
