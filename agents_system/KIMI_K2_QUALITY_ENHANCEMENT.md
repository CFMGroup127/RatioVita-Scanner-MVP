## 🎯 Quality Assessment Process

### **Step 1: Automatic Review**
**Trigger:** Agent marks task complete (P3 sign-off)

**Actions:**
1. Kimi K2 reads agent's memory document
2. Kimi K2 reviews all completed work
3. Kimi K2 assesses quality against real-world standards
4. Kimi K2 assigns quality score (0-100%)

### **Step 2: Quality Decision**

**Quality ≥ 80% (Excellent):**
- ✅ **Approve** work
- Log approval
- Notify agent
- Proceed to next phase

**Quality 70-79% (Good - Needs Improvement):**
- ⚠️ **Request Improvements**
- Create improvement checklist
- Assign rework tasks
- Provide specific guidance
- Set deadline
- Monitor progress

**Quality < 70% (Needs Significant Improvement):**
- ❌ **Reject** work
- Create detailed rework plan
- Assign rework with clear instructions
- Provide best practices guidance
- Escalate if needed
- Require re-submission
